# Лабораторна робота №4

## Візуалізація даних 2 — Інтерактивна гармоніка з шумом та фільтрацією

### Опис завдання

Інтерактивна програма для візуалізації гармоніки `y(t) = A * sin(ωt + φ)` з накладеним шумом та фільтрацією (Баттерворт lowpass).

### Можливості програми

- Слайдери для зміни параметрів гармоніки (амплітуда, частота, фаза)
- Слайдери для параметрів шуму (середнє, дисперсія)
- Слайдер для частоти зрізу фільтра
- Чекбокс для ввімкнення/вимкнення шуму
- Кнопка Reset для відновлення початкових параметрів
- Шум не перегенерується при зміні параметрів гармоніки
- Фільтр Баттерворта (lowpass) для відновлення чистої гармоніки

### Інструкція із запуску

**Вимоги:** Python 3.10+, pip

```bash
# 1. Створити віртуальне середовище
python -m venv venv

# 2. Активувати (Windows)
venv\Scripts\activate

# 3. Встановити залежності
pip install -r requirements.txt

# 4. Запустити програму
python lab4_harmonic.py
```

### Інструкція користувача

1. **Слайдери зліва** — змінюють параметри в реальному часі
2. **Amplitude** — амплітуда гармоніки (A)
3. **Frequency** — частота гармоніки (ω)
4. **Phase** — фазовий зсув (φ)
5. **Noise Mean** — середнє значення шуму
6. **Noise Covariance** — дисперсія шуму
7. **Cutoff Frequency** — частота зрізу фільтра Баттерворта
8. **Show Noise** — чекбокс для ввімкнення/вимкнення шуму
9. **Reset** — повертає всі параметри до початкових
